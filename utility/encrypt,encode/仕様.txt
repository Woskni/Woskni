AESEncrypt.cs
{
	文字列をAESを用いて暗号化・復号化します
	暗号化精度が高く、パスワードやセーブデータの暗号化に活用できます

	Encrypt:
		暗号化した文字列を返します

	Decrypt:
		復号化した文字列を返します
}

Base64Encode.cs
{
	文字列をBase64で符号化・復号化します
	AESに比べ高速ですが、暗号化としての性能は劣ります

	Encode:
		符号化した文字列を返します

	Decode:
		復号化した文字列を返します
}

RLE.cs
{
	文字列をRLE(ランレングス符号)で符号化・復号化します（例: "0001110110" => "033121"）

	Encode:
		符号化した文字列を返します

	Decode:
		復号化した文字列を返します
}

TextConverter.cs
{
	整数を漢数字表記の文字列に変換して返すためのクラスです
	
	漢数字は四種類から選ぶことが可能です（例：1205）
	・Normal		（例: 千二百五）
	・NormalDigitZero	（例: 一二〇五）
	・Old			（例: 阡弐佰伍）
	・OldDigitZero		（例: 壱弐〇伍）
}
